Приложение для учёта бюджета и ежедневных расходов.

## Запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/goldbergz/higher-web-practice-calculator.git

cd higher-web-practice-calculator
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск в режиме разработки

```bash
npm run dev
```

### 4. Сборка для продакшена

```bash
npm run build
```
